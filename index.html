Instalar en Windows

Se descargará un archivo .zip con todo lo necesario.

Pasos para instalar:
1. Extrae el ZIP en cualquier carpeta
2. Abre la carpeta extraída
3. Haz clic derecho en install.ps1
4. Selecciona "Ejecutar con PowerShell"

Windows puede pedir confirmación de seguridad — es normal al instalar fuera de la Microsoft Store.
